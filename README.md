# Python Chatbot

A simple console-based chatbot built with Python that interacts with users through greetings, name recognition, age calculation, counting, and a short programming quiz.

## Features

- Greets the user
- Asks for the user's name
- Calculates age using remainders
- Counts to any number provided by the user
- Includes a programming knowledge question

## Project Structure

- `main.py` → runs the chatbot
- `bot.py` → contains chatbot functions

## Technologies Used

- Python

## How to Run

```bash
python main.py
