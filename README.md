# 🤖 Python Chatbot

A fun console-based chatbot built with Python that interacts with users through greetings, name recognition, age guessing, counting, and a programming quiz!

## 💬 What Can It Do?

| # | Feature |
|---|---------|
| 1 | 👋 Greets the user |
| 2 | 📛 Asks for your name |
| 3 | 🎂 Guesses your age using remainders |
| 4 | 🔢 Counts to any number you want |
| 5 | 🧠 Tests your programming knowledge |

## 🚀 How to Run

1. Clone the repository:
```bash
git clone https://github.com/yourusername/python-chatbot.git
```

2. Navigate to the project folder:
```bash
cd python-chatbot
```

3. Run the program:
```bash
python main.py
```
